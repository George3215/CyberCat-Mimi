# 🐱 CyberCat-Mimi (简单版 / Simple Version)

这是 CyberCat 的初始单文件版本。

## 💡 特点
- **零依赖 (除 PyQt6)**：所有逻辑都在一个文件中，方便阅读和基础修改。
- **纯规则驱动**：猫咪的行为完全基于随机数和预设概率，不占用 AI 算力。
- **极简部署**：只需运行 `python cyber_cat.py` 即可启动。

## 🚀 启动方式
```bash
python simple_version/cyber_cat.py
```
