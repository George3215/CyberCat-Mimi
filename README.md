# 🐱 CyberCat Mimi - 桌面可爱猫咪宠物

这是一个专为 macOS 设计的轻量级桌面电子宠物项目，使用 Python 和 PyQt6 编写。让一只可爱的 8-bit 像素猫咪在你的屏幕上自由漫步、休息或追逐蝴蝶，陪伴你的每一天。

## ✨ 特性

- **多样化行为**：猫咪会自动切换状态，包括静坐（Sit）、奔跑（Run）以及独特的“追逐蝴蝶”（Chase Butterfly）行为。
- **macOS 全面适配**：
  - **窗口置顶**：始终保持在屏幕最前端。
  - **多桌面支持**：当你在不同的 macOS 虚拟桌面（Spaces）或者全屏应用之间切换时，猫咪依然如影随形。
  - **点击穿透**：猫咪窗口不会干扰你的正常鼠标操作，你可以直接点击它下方的窗口。
- **精致的设计**：
  - **透明背景**：无边框设计，完美融入你的桌面。
  - **8-bit 像素风**：复古可爱的视觉效果。
  - **物理反馈**：猫咪在跑出屏幕边缘时会智能转向。

## 🚀 快速开始

### 前置条件

确保你的系统中已安装 **Python 3.x**。

### 安装依赖

项目依赖于 `PyQt6`。你可以通过 pip 进行安装：

```bash
pip install PyQt6
```

### 运行程序

在终端中进入项目目录，运行：

```bash
python cyber_cat.py
```

## 📂 项目结构

- `cyber_cat.py`: 核心逻辑代码，包含状态机、窗口管理及绘图逻辑。
- `Cats Download/`: 存放各种猫咪品种的 8-bit 像素素材。目前默认使用 `black_2.png`。

## 🎨 自定义

你可以通过修改 `cyber_cat.py` 中的 `IMG_PATH` 变量来更换不同颜色的猫咪：

```python
# 修改此处路径即可更换猫咪
IMG_PATH = "Cats Download/orange_0.png"
```
